# 禁用/取消禁用猎头

可根据猎头 ID 对猎头执行禁用/取消禁用操作。被禁用的猎头，不能推荐候选人与被分配职位。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=hire&version=v1&resource=agency&method=operate_agency_account)

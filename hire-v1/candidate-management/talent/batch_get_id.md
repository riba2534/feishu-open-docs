# 批量获取人才ID

通过手机号、邮箱、证件号，批量查询人才ID{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=hire&version=v1&resource=talent&method=batch_get_id)

:::html
<md-alert type="error">

</md-alert>
:::

:::html
<md-alert type="warn">
- 入参的手机号、邮箱、证件号至少需要传一种，否则无法查询到人才数据
- 入参的手机号、邮箱、证件号是AND的逻辑关系。也就是当条件同时满足时才会返回人才数据。
</md-alert>

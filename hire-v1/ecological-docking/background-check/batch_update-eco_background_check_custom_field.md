# 更新背调自定义字段

更新用户在发起背调时展示的表单自定义字段名称和描述。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=hire&version=v1&resource=eco_background_check_custom_field&method=batch_update)

:::html
<md-alert type="error">

</md-alert>
:::

:::html
<md-alert type="warn">
* 更新操作将影响已发起的背调表单展示，但不影响实际背调内容。
* 更新操作只允许更新已有字段的名称和描述，不支持更新字段类型，也不允许新增或删除自定义字段。
</md-alert>
:::

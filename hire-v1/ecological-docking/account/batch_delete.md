# 删除账号自定义字段

飞书招聘的背调或笔试服务商，可通过此接口删除账号自定义字段（如客户在服务商处的租户 ID、账号 ID等）。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=hire&version=v1&resource=eco_account_custom_field&method=batch_delete)

:::html
<md-alert type="error">

</md-alert>
:::

:::html
<md-alert type="warn">
* 删除操作不支持撤销，对应的自定义字段的 key 将无法重复使用。
* 该接口不支持删除所有的自定义字段——即须保留至少一个自定义字段。如需删除所有字段，可先通过[创建账号自定义字段](/ssl:ttdoc/ukTMukTMukTM/uMzM1YjLzMTN24yMzUjN/hire-v1/eco_account_custom_field/create)接口新增，然后将多余的自定义字段删除。
* 删除后，客户已有账号的自定义字段的值不受影响，只影响添加新账号时的表单展示。
</md-alert>
:::

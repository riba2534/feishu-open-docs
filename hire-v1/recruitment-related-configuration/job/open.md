# 重启职位

对于已关闭的职位，可通过本接口重启职位。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=hire&version=v1&resource=job&method=open)

:::html
<md-alert type="error">

</md-alert>
:::

:::html
<md-alert type="warn">

</md-alert>
:::

:::html
<md-alert type="tip">
在调用本接口前，须在「飞书招聘」-「设置」-「职位管理」-「职位设置」中开启「通过 API 同步职位」开关，否则将只能在招聘系统内重启职位。
</md-alert>
:::

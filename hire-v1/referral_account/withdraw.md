# 全额提取内推账户余额

通过账户 ID 全额提取内推账户下的积分/现金。全额提现后，内推人在飞书招聘系统中的积分/现金余额会变为 0，对应的积分/现金奖励状态也会变为「已发放」。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=hire&version=v1&resource=referral_account&method=withdraw)

# 查询职等信息

查询职等的详细信息。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=corehr&version=v2&resource=job_grade&method=query)

:::html
<md-alert type="tip">
- 每次最多传 100 个职等 ID 和 Code，如果不传则默认无筛选条件，返回全部列表
</md-alert>
:::

:::html
<md-alert type="warn">
- 延迟说明：数据库主从延迟 2s 以内，即：直接创建职等后2s内调用此接口可能查询不到数据。
- 所有筛选项可一起使用，之间为 AND 关系
</md-alert>
:::

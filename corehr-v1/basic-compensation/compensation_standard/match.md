# 获取员工薪资标准

- 调用此接口来获取员工匹配的薪资标准信息
- 请求时，可选传递员工的部门 ID、职级 ID、职务ID 、序列 ID 等筛选条件，用于匹配薪资标准
- 此接口将返回员工可匹配到的薪资标准全部信息，包括薪资标准表 ID、薪级薪等、薪资带宽、薪资标准值等{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=corehr&version=v1&resource=compensation_standard&method=match)

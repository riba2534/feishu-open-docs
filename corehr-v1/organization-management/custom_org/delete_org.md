# 删除自定义组织

根据传入的自定义组织 ID 删除相应自定义组织{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=corehr&version=v2&resource=custom_org&method=delete_org)

:::html
<md-alert type="tip">

</md-alert>
:::

:::html
<md-alert type="warn">

</md-alert>
:::

:::html
<md-alert type="error">
- 删除后无法恢复， 并且在系统中无法搜索到对应自定义组织信息，请谨慎操作。
- 删除自定义组织时请确认有无员工、异动单据、待入职单据关联此自定义组织，如有会导致删除失败。
</md-alert>
:::

# 批量查询部门操作日志

批量查询指定时间范围内的部门操作日志{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=corehr&version=v2&resource=department&method=query_operation_logs)

:::html
<md-alert type="tip">
- 默认排序条件：默认以操作时间倒序排序
- 仅支持查询部门基础字段变更的操作日志，对于角色字段变更的操作日志查询功能待上线

</md-alert>
:::

:::html
<md-alert type="warn">
- 时间窗支持大范围查询，限定查询范围在366天以内，例如要查询2020年1月1日至2023年1月1日的数据，建议分成两次查询，分别为2020年1月1日至2021年1月1日，2022年1月2日至2023年1月1日，不建议查询大时间范围数据
- 支持查询批量部门的操作日志，限定查询最大部门数为100，查询部门数量大于100时，建议分批查询
</md-alert>
:::

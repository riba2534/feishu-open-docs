# 批量分页查询地点信息

该接口用于批量查询地点信息，信息包含地点名称、描述、地点用途、工时制度、区域设置、时区以及关联的地址信息等。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=corehr&version=v1&resource=location&method=list)

:::html
<md-alert type="error">

</md-alert>
:::

:::html
<md-alert type="warn">
延迟说明：数据库主从延迟 2s 以内，即：直接创建地点后2s内调用此接口可能查询不到数据。
</md-alert>
:::

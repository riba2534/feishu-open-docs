# 获取词典分类

获取飞书词典当前分类。<br>
飞书词典目前为二级分类体系，每个词条可添加多个二级分类，但选择的二级分类必须从属于不同的一级分类。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=baike&version=v1&resource=classification&method=list)

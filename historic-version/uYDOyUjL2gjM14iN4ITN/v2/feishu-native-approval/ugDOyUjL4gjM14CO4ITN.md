### 请求体

| 名称         | 类型           | 必须        | 说明        |
| --------- | --------------- | -------   | --------- |
|approval_code | string | 是 |  审批定义唯一标识 |

### 请求体示例

```json

	"approval_code":"7C468A54-8745-2245-9675-08B7C63E7A85"
}
````

```json
}
```

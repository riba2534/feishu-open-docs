# 获取关联组织详情

基于组织tenant key获取关联组织详情，需要对对方组织有可见权限才可以获取。{尝试一下}(url=/api/tools/api_explore/api_explore_config?project=trust_party&version=v1&resource=collaboration_tenant&method=get)

## 提示
使用 user_access_token 时，按照 admin 管理后台关联组织列表中针对用户设置的可见性规则进行校验，使用 tenant_access_token 时，按照应用互通界面中针对应用设置的可见性规则进行校验。
